BPlayer Privacy Policy

Last updated: July 2026

Data Collection

BPlayer does not collect, store, or transmit any personal data. All video files, metadata, and configuration are stored locally on your device.

Network Access

BPlayer may make network requests to fetch metadata (e.g., movie posters, descriptions) from configured API endpoints. These requests are initiated only when you enable online metadata lookup in Settings. No usage data is sent to any server controlled by the developer.

Third-Party Services

BPlayer uses LibVLC (VideoLAN) for media playback. No data is shared with VideoLAN beyond standard library calls.

Local Storage

BPlayer stores app configuration, theme preferences, and cached thumbnails in your local app data folder. This data never leaves your device.

Changes

This policy may be updated. Check this page for the latest version.
